Recaptcha_XH - a CMSimple_XH Plugin

Recaptcha_XH is a captcha plugin powered by reCAPTCHA. It could be used as utility plugin for other plugins, such as Advancedform_XH (Version 1beta7 and up).

Requirements

Recaptcha_XH is a plugin for CMSimple_XH. It requires an UTF-8 encoded version. To use it, you have to sign in with a Google Account to get the required keys.

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory recaptcha to your server into the CMSimple_XH plugins directory.
  4. Set write permissions to the subdirectories config/ and languages/.
  5. Switch to Recaptcha in the back-end to check if all requirements are fulfilled. Missing requirements will be reported as a warning. You should not proceed using Recaptcha_XH until all it's requirements are met, as further checks are avoided for performance reasons, and that might lead to faulty behaviour.

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select Recaptcha from "Plugins".

You can change the default settings of Recaptcha_XH in "Plugin Config". Hints for the options will be displayed when hovering over the help icon with your mouse. The most important settings are the keys. To get the required keys for reCAPTCHA you have to create a Google Account, if you do not already have one. Then you have to sign up and create the keys.

Localization is done in "Plugin Language". You can translate the character strings to your own language, or customize them according to your needs. Somewhat special is the group "captcha". This is the localization of reCAPTCHA itself. For the languages en, nl, fr, de, pt, ru, es and tr reCAPTCHA already provides localization, so the entries could be left blank.

Currently no stylesheet is distributed with Recaptcha_XH as it's not clear, if this is really necessary, because there are 4 different themes already available. And it might be difficult to use the stylesheet in all circumstances, as reCAPTCHA will be integrated in an IFrame, if Javascript is disabled in the visitors browser.

Usage

Recaptcha_XH is a utility plugin serving other plugins. As such it has no usage on it's own.
